## Features 

* Hot Module Reload
* Pretty easy to get up and running
* Simple webpack config ([See here](./webpack.config.js))

### Setup and Run

* Clone this repo
* Type ```npm run clean-commits``` // to start with a fresh git repo 
* Type ```npm install```
* Type ```npm run dev```
* In the browser,  type ```localhost:5000```
